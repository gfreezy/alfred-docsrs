# alfred-docsrs
alfred-docsrs is a workflow to search [https://docs.rs](https://docs.rs).

# Commands
## Search a crate or a symbol in a crate:

```
docsrs cratename symbol_name
```

## Clear cache
```
docsrs-clear
```
